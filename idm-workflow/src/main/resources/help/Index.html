IDM

Introduction

IDM is an Analysis Platform that runs in parallel with the Hadoop platform and other Hadoop Technologies.

Menu

The Menu contains basic operations for manipulating workflow and projects

Canvas

The Canvas contains a Workflow that is used in IDM to manage a jobs processes or flow. A worklow is a build up of processes that chain together that perform corrective actions that produce a desired output. It is a way of managing a job so that each aspect of the job can be modified to use desired parameters

Actions Footer

Actions footer is a container for tabs that hold actions that can be added to the workflow. Tabs can be created and be organized to hold certain actions.
