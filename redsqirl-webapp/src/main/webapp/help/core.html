Core

Description

Core gathers utility functions available out of the box.

  • Sources: Generic dataset declaration
  • Email: Send email after an action is completed
  • Convert: Convert a directory to a file.
  • Script: Runs a script in the language of your choice
  • Superaction: Encapsulate complexity in a super action
  • Schedule: HDFS or HCatalog Source and Sink for building data pipelines
