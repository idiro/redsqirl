Create a bespoke script

  1. Create the oozie action xml extract & script in the script action and make sure that it runs.
  2. Create a directory with no special characters but dash and underscore. This directory can be located in a share folder, RS_HOME/scripts, so that other users can use your script or in your user folder, RS_HOME/users/MY_USER/scripts.
  3. Copy the action xml in a file called oozie-action.xml and your script in script
  4. Create a script.properties file
    • script.extension the script extension, it is mandatory if you have a script
    • output.names is a coma delimited list with the output name, note that if the list is empty, it defines one output with an empty name
    • output.type type of the output to create. For example TEXT MAP-REDUCE DIRECTORY, you can find this value if you created a script in the Format page. If you have named output, you should define output.type.NAME.
    • output.header header with types of the output. You can find this value if you created a script in the Header page. If you have named output, you should define output.header.NAME.
    • input.number.min - optional validation - the minimum number of input expected
    • input.number.max - optional validation - the maximum number of input expected
    • input.header - optional validation - the header that all the inputs should have
  5. Create a file called bespoke
