Help

Description

Displays help for Actions , Tabs , Views and Menus. Each help item is automatically displayed when the item is selected on Red Sqirl. This feature can be disabled on the top of the view, alongside other buttons to navigate around the Help View.

  • Previous - Go to the previous page
  • Forward - Go to the page that was previously viewed
  • Stay on the page - Prevent the help to change page
  • Index - Go to the help index page
