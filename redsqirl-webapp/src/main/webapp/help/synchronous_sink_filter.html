Synchronous Sink Filter

Description

Synchronous Sink Filter is an action that reuse a Sink output into another coordinator job. This node allows data pipeline in one workflow.

Quick Guide

  • Inputs - The Synchronous Synk output.
  • Template path - The path in which the data is stored
  • Outputs - The selected instances

Path Selection - Page 1

  • Number of instances: The number of the most recent instances to use for your analyses
  • Offset: The workflow runs at time t with a dataset timestamp labeled t+n*freq. A negative number means a delay between the data set and the running.
