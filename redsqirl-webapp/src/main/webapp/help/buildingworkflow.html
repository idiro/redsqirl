Building a Workflow

The initial step in builidng a workflow is to add/drag a Source to the canvas , from there the source configuration must be complete. The Source Help File shows step by steps

Adding the Source and configuring it means that actions can be added and linked to the Source. Depending on the action more that one source may be needed. Read each action help file to learn how to use the action.

Actions can be linked together depending on the action type and output. This will lead to building workflows that may be quicker that running multiple separate workflows.

When the workflow is ready to run , the menu holds the option to run the project under Projects
