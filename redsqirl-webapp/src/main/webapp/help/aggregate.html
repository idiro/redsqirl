Aggregate

Aggregate is the easiest way to create Super Actions. Super Actions are a logical group of actions, they have a name and are included into a model. Super Actions can be reused in any other workflow and included into the user's footer.

Aggregate will always install the element in the user scope. You can choose the model name under which the Super Action is installed. In a Super Action inputs and outputs have to be declare, their name have to unique and smaller than 15 characters. A short comment can be added optionnaly to guide future uses.

In order to modify a Super Action, the easiest way is to:

  1. Drag and drop the action on a canvas.
  2. Hover the Super Action, Options>Edit Super Action.
  3. Do the modifications you want, it can include more comments in the Comment Editor.
  4. Go into Software>Install Sub-workflow
