WHAT IS RED SQIRL?

Red Sqirl is a web-based big data application that simplifies the analysis of large data sets. With Red Sqirl, you can quickly and cost-effectively access the power of the Hadoop eco-system, enhancing the productivity of data scientists and analysts.

If you want to analyse large data sets in the most efficient way possible, Red Sqirl is the answer.

What can Red Sqirl do?

• Analyze massive amounts of data rapidly
and cost effectively.

• Also available on the cloud so no need
to outlay budget for hardware.

• Provides additional processing horsepower at
the press of a button.

• Open platform that users can extend.

• Simplifies the Hadoop eco-system (Hadoop,
Hive, Pig, HBase, Oozie, etc) so you don't
have to master each of those underlying
technologies.
