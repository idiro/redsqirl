Source

Description

Source is an action that defines a data source for a workflow. A data source can come from a type of sorage facility. There are two types of storage facilities: Hive and HDFS.

Data Type - Page 1

Data type is where the data storage facility is chosen. As mentioned before the only options are Hive and HDFS.

Hive

Hive is a relational database that is used on Hadoop. It has a query language that can used like any other SQL.

HDFS

HDFS is the file system that is used on Hadoop. HDFS stands for Hadoop Distributed File System.

Data Sub-Type - Page 2

With each data type there are data sub-types. These are specific to the parent (data type)

Hive

Hive Table

A Hive table is table that is stored in Hive and is similar to the table that is stored in other RMDB (Oracle)

Hive Partition

A Hive partition is a similar to a table except it is stored so that partition can be filtered based on a condition (i.e income greater that 50000)

HDFS

Data File

A Data File is the specific file that is to be used as the data source

Text Map Reduce Directory

Text Map Reduce Directory is the directory that is to be used that contains the data. The directory will contain all the data that a map reduce job will output

Binary Map Reduce Directory

Binary Map Reduce Directory is a directory of a map reduce job that stores the values in format.
