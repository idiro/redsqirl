Convert

Description

Convert a dataset from one format to another. At the moment only Hive Table and Map-Reduce Text directories are supported.

Output Format - Page 1

Output

Set the output format of the convert. This depends on the input type. For example , if the input type is hive then the output format will be HDFS directory type (Just Text for the moment)
