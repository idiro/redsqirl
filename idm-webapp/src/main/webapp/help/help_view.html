Help

Description

Displays help for Actions , Tabs , Views and Menus. Each help Item is automatically displayed when the Item is selected on IDM. This feature can be disabled on the top of the view along with other buttons to navigate around the Help View.

  • Previous - Go to the previous page
  • Forward - Go to the page that was previously viewed
  • Don't Follow the links - Stop the help page from
  • Index - Go to the help index page
